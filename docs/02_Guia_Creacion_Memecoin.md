# Guía de Creación del Memecoin BlueWave (BLUEW)

## Datos del Token

| Campo | Valor |
|---|---|
| **Nombre** | BlueWave |
| **Ticker** | BLUEW |
| **Blockchain** | Solana |
| **Suministro total** | 1,000,000,000 (mil millones) |
| **Tipo de lanzamiento** | Fair Launch (sin pre-sale) |
| **Impuesto transacciones** | 2% (100% a marketing) |
| **Red social logo** | Ola azul + Barika Hel + texto "BlueWave" |

---

## Paso 1: Crear Wallet Phantom

1. Ve a [phantom.app](https://phantom.app) e instala la extensión de navegador (Chrome/Brave/Firefox)
2. Haz clic en "Create a new wallet"
3. **Guarda la frase semilla (seed phrase) en papel. NUNCA en digital.**
4. Establece una contraseña segura
5. Anota tu dirección pública (empieza con una letra, ej: `Gx8...`)

> **💡 Crea UNA wallet dedicada para el proyecto. No uses tu wallet personal.**

## Paso 2: Obtener SOL para comisiones

Necesitas un poco de SOL (cripto nativa de Solana) para pagar comisiones:
- Compra SOL en un exchange (Binance, Kraken, Coinbase)
- Transfiere ~0.1 - 0.5 SOL a tu wallet Phantom (suficiente para lanzar)
- Guarda el SOL restante para promociones

## Paso 3: Elegir plataforma de lanzamiento

### Opción recomendada para principiantes: **Pump.fun**
- URL: [pump.fun](https://pump.fun) (verifica siempre la URL oficial)
- No requiere código de smart contract
- Plataforma diseñada para memecoins en Solana
- El token se crea y automáticamente tiene liquidez

### Alternativa avanzada: Solana CLI + Meteora
- Requiere conocimientos de línea de comandos
- Más control sobre el token
- Crearás un SPL Token manualmente

**Recomendación para ti:** Usa **Pump.fun** para el lanzamiento inicial.

## Paso 4: Crear el token en Pump.fun

1. Conecta tu wallet Phantom a pump.fun
2. Haz clic en "Create Coin"
3. Completa:
   - **Name:** BlueWave
   - **Ticker:** BLUEW
   - **Description:** "BlueWave - Good vibes & tourism from Curaçao. Representing the heart of Curaçao culture and its beautiful Barika Hel bird. $BLUEW"
   - **Image:** Sube el logo (ola azul + Barika Hel + texto "BlueWave")
   - **Supply:** 1,000,000,000
4. Confirma la transacción (pagas ~0.02 SOL de comisión)

## Paso 5: Verificar el token

- Busca tu token en [solscan.io](https://solscan.io) pegando la dirección del token
- Confirma que el suministro, nombre y ticker sean correctos
- Comparte la dirección del token en tus redes

## Paso 6: Crear liquidez (si no usas Pump.fun)

Si usas Pump.fun, la liquidez se crea automáticamente. Si usas otro método:
1. Usa Meteora o Raydium para crear un pool de liquidez
2. Añade una cantidad de SOL y BLUEW al pool
3. Esto permite que la gente pueda comprar/vender

## Paso 7: Distribución inicial

Al ser Fair Launch:
- Todo el suministro va al pool de liquidez
- No hay equipo ni pre-venta
- Tú compras tus propios tokens como cualquier persona

## Paso 8: Configurar el impuesto (tax)

Si usas Pump.fun, el 2% de tax se configura en el contrato. Algunas opciones:
- **Simple tax:** 2% a una wallet de marketing (que tú controlas)
- En Pump.fun esto varía — investiga la opción "tax" en la creación
