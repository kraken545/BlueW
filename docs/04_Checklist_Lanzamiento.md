# Checklist Completo de Lanzamiento - BlueWave (BLUEW)

## Fase 1: Preparación (Semana 1-2)

### Cuentas y Perfiles
- [ ] Crear cuenta **Twitter/X**: `@BlueWaveCuracao` (o similar)
- [ ] Crear **página de Facebook**: "BlueWave Curaçao"
- [ ] Crear **cuenta de Instagram**: `@bluewave_curacao`
- [ ] Crear **canal de YouTube**: "BlueWave Curaçao"
- [ ] Crear **servidor de Discord**: "BlueWave Community"
- [ ] Crear **cuenta de TikTok**: `@bluewavecuracao`
- [ ] Unificar nombre de usuario en todas las plataformas (ej: `bluewave_curacao` o `@BlueWaveCuracao`)

### Blockchain y Wallet
- [ ] Instalar **Phantom Wallet** en navegador
- [ ] Descargar **Phantom Wallet** en móvil
- [ ] Guardar frase semilla en papel (3 copias en lugares seguros)
- [ ] Comprar y transferir ~0.2 SOL a la wallet del proyecto
- [ ] No conectar la wallet a sitios desconocidos

### Logo y Branding
- [ ] Generar logo con IA (prompt en documento 02)
- [ ] Crear banner/banner para Twitter, YouTube, Facebook
- [ ] Crear icono cuadrado para perfiles
- [ ] Crear fondo para Discord

### Contenido Inicial
- [ ] Escribir "bio" para cada red social
- [ ] Crear 10 posts/tweets de respaldo listos para publicar
- [ ] Grabar 1 video promocional corto (15-30 seg) sobre Curaçao + BlueWave

---

## Fase 2: Infraestructura Técnica (Semana 2-3)

### Automatización
- [ ] Crear cuenta en **Oracle Cloud** (free tier)
- [ ] Crear instancia **VM.Standard.A1.Flex**
- [ ] Acceder por SSH
- [ ] Instalar **Node.js + n8n**
- [ ] Configurar acceso web a n8n (puerto 5678)
- [ ] Crear flujo de publicación diaria en Twitter
- [ ] Conectar API de Facebook
- [ ] Conectar API de Instagram
- [ ] Conectar API de YouTube
- [ ] Configurar webhook de Discord
- [ ] Conectar API de TikTok
- [ ] Probar publicación en cada red

### Creación del Token
- [ ] Decidir plataforma (Pump.fun recomendado)
- [ ] Conectar wallet a plataforma
- [ ] Crear token **BlueWave (BLUEW)**
- [ ] Verificar token en Solscan.io
- [ ] Configurar impuesto 2% a wallet de marketing

---

## Fase 3: Pre-Lanzamiento (Semana 3-4)

### Marketing y Comunidad
- [ ] Publicar primer tweet anunciando BlueWave
- [ ] Crear Discord con canales: #general, #anuncios, #precios, #memes
- [ ] Invitar primeros miembros (amigos, comunidades de Curaçao)
- [ ] Publicar video teaser en YouTube/TikTok
- [ ] Empezar automatización de contenido diario
- [ ] Investigar hashtags populares de Curaçao y memecoins

### Seguridad
- [ ] Confirmar que nadie tiene acceso a la wallet de marketing
- [ ] Revocar permisos innecesarios de smart contract
- [ ] No hacer clic en enlaces sospechosos
- [ ] No compartir seed phrase con nadie

---

## Fase 4: Lanzamiento (Día 0)

- [ ] Hacer último test en devnet (si aplica)
- [ ] Tener SOL suficiente para comisiones
- [ ] Publicar en todas las redes: "BLUEW ya está en vivo!"
- [ ] Compartir dirección del token en todas las plataformas
- [ ] Publicar en sitios de tracking de memecoins (CoinGecko, DexScreener)
- [ ] Monitorear primeras transacciones
- [ ] Responder preguntas en Discord/Telegram

---

## Fase 5: Post-Lanzamiento

- [ ] Publicar contenido mínimo 1 vez al día (automatizado)
- [ ] Crear comunidad activa en Discord
- [ ] Hacer sorteos/concursos de tokens
- [ ] Colaborar con influencers de Curaçao
- [ ] Mantener automatización activa 24/7
- [ ] Revisar y ajustar impuesto si es necesario
- [ ] Nunca compartir llaves privadas
