# Guía Básica de Blockchain para Principiantes

## 1. ¿Qué es Blockchain?

Una **blockchain** (cadena de bloques) es un libro de contabilidad digital, público e inmutable. Imagina un cuaderno donde cada página es un "bloque" de transacciones. Una vez que se escribe una página, no se puede borrar ni modificar, y todo el mundo tiene una copia del cuaderno.

- **Público:** Cualquiera puede ver las transacciones.
- **Inmutable:** Una vez escrita, nadie puede cambiarla.
- **Descentralizado:** No depende de un solo servidor o empresa.

## 2. ¿Qué es una Criptomoneda?

Es un activo digital que vive en una blockchain. Ejemplos: Bitcoin, Ethereum, Solana. Un **memecoin** es una criptomoneda inspirada en memes, cultura o conceptos virales (Ej: Dogecoin). BlueWave es un memecoin.

## 3. ¿Qué es Solana?

**Solana** es una blockchain rápida y barata:
- Transacciones en segundos
- Comisiones de ~$0.0002 por transacción
- Ideal para memecoins por su bajo costo

## 4. Conceptos Clave

| Concepto | Explicación simple |
|---|---|
| **Wallet** | Cartera digital para guardar tus tokens. Como una cuenta de banco pero tú controlas las llaves. |
| **Llave privada** | Una contraseña secreta que NUNCA debes compartir. Quien la tiene, controla tus tokens. |
| **Llave pública / Dirección** | Es como tu número de cuenta. La compartes para recibir tokens. |
| **Smart Contract** | Un programa que vive en la blockchain. Define las reglas de tu memecoin. |
| **Token** | La moneda en sí. Cada unidad se llama token. |
| **Suministro total** | Cantidad total de tokens que existirán. BlueWave: 1,000,000,000. |
| **Liquidez** | Dinero en un "pool" para que la gente pueda comprar/vender tu token. |
| **Fair Launch** | Lanzamiento sin pre-venta. Todos compran al mismo tiempo desde el inicio. |

## 5. Seguridad - PUNTOS CRÍTICOS

### ✅ HACER
- Guardar la llave privada **OFFLINE** (escrita en papel, NUNCA en digital)
- Usar una wallet dedicada SOLO para el proyecto
- Verificar URL de sitios antes de conectar tu wallet
- Probar todo en **red de prueba (devnet)** antes de lanzar en **mainnet** (red real)

### ❌ NUNCA HACER
- Compartir tu llave privada con NADIE (ni soporte, ni amigos)
- Hacer clic en enlaces sospechosos de "airdrop" o "sorteos"
- Conectar tu wallet a sitios no verificados
- Almacenar llaves privadas en fotos, documentos en la nube o capturas de pantalla

### 🛡️ Buenas prácticas
- Crea una wallet exclusiva para el proyecto (no mezcles con tu wallet personal)
- Usa una **ledger (hardware wallet)** si el proyecto crece
- No compartes la frase semilla (seed phrase) con nadie
- Revoca permisos de contratos viejos (puedes usar solscan.io para revisar)
