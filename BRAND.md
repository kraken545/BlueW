# BlueWave Brand Guide

## Color Palette

| Color | Hex | Usage |
|---|---|---|
| Deep Teal | `#0077B6` | Primary brand color, buttons, headings |
| Dark Teal | `#005A8C` | Dark backgrounds, footer |
| Light Teal | `#00B4D8` | Accent, gradients |
| Turquoise | `#48CAE4` | Light accents |
| Sky | `#90E0EF` | Subtle backgrounds |
| Sunrise Yellow | `#FFD200` | CTA buttons, highlights, ticker |
| Coral | `#FF6B00` | Hover states, warm accents |
| Warm Sand | `#F4E4C1` | Card backgrounds |
| Cream | `#FFF8F0` | Page background |
| Text Dark | `#1A1A2E` | Body text |

## Typography

- **Display:** Playfair Display (600 weight) — for titles, hero text
- **Body:** DM Sans (400, 500, 600, 700) — for all other text
